# About Me

I am a **Systems-focused Software Engineer** with a deep interest in **backend engineering, distributed systems, concurrency, and scalable software architecture**.

Rather than treating programming as writing features, I enjoy treating it as **designing reliable systems**—breaking large engineering problems into predictable, maintainable components that continue to perform under growth and failure.

My foundation comes from **competitive programming**, where algorithmic thinking became second nature. That analytical mindset now extends into backend design, asynchronous processing, database optimization, and distributed computing.

I believe software should remain elegant even under complexity.

## 🔎 Distributed Search Engine

**Java • Spring Boot • MySQL • Redis • CompletableFuture**

A custom distributed search engine exploring how modern search infrastructure works internally.

```text
Crawler
    │
    ▼
Tokenizer
    │
    ▼
Inverted Index
    │
    ▼
Shard Manager
    │
    ▼
Search Nodes
    │
    ▼
Ranked Results
```

**Highlights**

- ⚡ Scatter-Gather query execution
- ⚡ Concurrent processing with `CompletableFuture`
- ⚡ BFS Web Crawler using Jsoup
- ⚡ Hash-based sharding
- ⚡ Primary-Replica architecture
- ⚡ Heartbeat failure detection
